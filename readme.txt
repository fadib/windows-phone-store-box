WordPress plugin to display windowsphone.com store app information in a box inside post or page.

== Description ==

== Installation ==

For help installing this (or any other) WordPress plugin, please read the [Managing Plugins](http://codex.wordpress.org/Managing_Plugins) article on the Codex.

**Step 1)** After installing the plugin, go to the WPS Box > Settings screen and select the country of the store you want to use.

**Step 2)** [Add the [wpsbox] shortcode to a post or page](http://codex.wordpress.org/Shortcode), and include the application id you want to show.

Examples:

`[wpsbox id="/pandora/de2df279-485d-49bb-b53e-3f6a2a9401c1"]`

`[wpsbox id="/flixster/7dc02baf-a7d6-df11-a844-00237de2db9e"]`
